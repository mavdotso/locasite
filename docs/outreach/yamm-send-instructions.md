## Step 2 — Create the Gmail Draft

YAMM works by merging a Gmail draft. Create one draft that uses merge markers.

**Important:** Since every row has a fully pre-written `Email Body`, you have two options:

### Option A — Full body merge (recommended for this batch)

Create a Gmail draft with just the merge marker in the body:

```
{{Email Body}}
```

Set the subject line to:
```
{{Email Subject}}
```

This will use the exact personalized body and subject line from each row.

### Option B — Send as-is without merge

If YAMM's multi-line body support is unreliable, alternatively:
- Use YAMM in "one draft per recipient" mode
- Or send manually from the draft file using Gmail's BCC workaround (not recommended for cold outreach)
