# Locosite — Roadmap

## Current Status: ~75% Ready for Alpha

### ✅ Done
- Core site generation from Google Maps
- Subdomain routing middleware
- Convex integration (database, auth with GitHub)
- Visual editor (drag-and-drop blocks, columns, inline editing)
- Simple builder (section-based, business presets)
- Business claiming flow
- Stripe payment integration
- Analytics (Tinybird + Convex hybrid)
- Landing page redesign
- Auth redirect flow fixed
- Build passes, lint clean

### 🟡 Remaining for Alpha
1. **CI/CD pipeline** — automated testing + deployment
2. **Staging environment** — test before prod
3. **Error monitoring (Sentry)** — catch issues in production
4. **Performance monitoring** — track load times

### 🟠 Post-Alpha Features
- Mobile-responsive preview + editing
- AI content generation
- Multi-language support
- Email/messages integration
- Team management
- Version history with rollback

### 🔵 Backlog
- Component templates library
- Advanced analytics (cohorts, funnels)
- Data export (CSV, JSON)
- E2E tests (Playwright/Cypress)

## Estimated Completion
- **Alpha release**: ~1-2 weeks (needs CI/CD + monitoring)
- **Beta with features**: ~4-6 weeks
